# OpenAiCodexTestprojekt

Dies ist ein einfaches Beispiel, das eine SWT-Anwendung mit einer Menüsstruktur zeigt. Die Anwendung kann mit dem folgenden Befehl kompiliert werden:

```bash
javac -classpath /usr/share/java/swt4.jar -d out src/com/example/VersionApp.java
```

Zum Starten verwendet man:

```bash
java -classpath /usr/share/java/swt4.jar:out com.example.VersionApp
```

Die Version der Anwendung wird im Fenstertitel und im „Über“-Dialog angezeigt.
